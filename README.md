
# HowTo - Understanding AI and Data Utilization, Gamification with Game 

원광대학교 게임콘텐츠학과 이용환 교수의 **고교특강 연계강좌**  
**“게임으로 이해하는 인공지능·데이터 활용과 게임화”** 강의용 GitHub 저장소입니다.

이 저장소는 고등학생들이 게임을 매개로 인공지능, 데이터 활용, 데이터 분석, 데이터 시각화, 게임화(gamification)의 개념을 쉽고 흥미롭게 이해할 수 있도록 지원하기 위해 제작되었습니다.  
강의자료, 활동 예시, 탐구보고서 작성 자료, 발표 참고자료 등을 체계적으로 정리하여, 수업 중 활용과 수업 후 복습이 모두 가능하도록 구성합니다.

## 🏆 Program Purpose

본 프로그램은 게임콘텐츠 분야에서 실제로 활용되는 인공지능과 데이터 분석 사례를 바탕으로,  
고등학생들이 인공지능과 데이터 활용의 기본 원리를 **쉽고 흥미롭게 이해**할 수 있도록 설계된 연계 강좌입니다.

학생들은 다음과 같은 내용을 경험하게 됩니다.

- 게임 속 인공지능 작동 원리 이해
- 이용자 행동 데이터와 플레이 패턴 분석
- 데이터 수집·분류·비교·해석의 기초 이해
- 그래프와 차트를 활용한 데이터 시각화
- 게임화를 활용한 실생활 문제 해결 아이디어 탐구
- 팀 기반 탐구보고서 작성 및 발표 경험
- AI·데이터·게임 관련 진로 탐색

즉, 이 프로그램은 단순한 지식 전달을 넘어서  
**게임을 통해 AI와 데이터를 이해하고, 이를 실생활 문제 해결 아이디어로 확장하는 융합형 학습 과정**을 목표로 합니다.

## 🛠️ Educational Goals

이 프로그램의 주요 목표는 다음과 같습니다.

1. 게임 사례를 통해 인공지능의 기본 개념을 이해한다.
2. 게임에서 데이터가 어떻게 수집·분석·활용되는지 설명할 수 있다.
3. 데이터 분석과 시각화의 기초 방법을 익힌다.
4. 게임화 개념을 이해하고 실생활 문제 해결 아이디어로 확장할 수 있다.
5. 팀 활동과 발표를 통해 협업 능력과 표현 능력을 기른다.
6. 미래 사회에서 AI·데이터·게임 기술이 가지는 의미를 이해하고 진로와 연결해 생각해 본다.

## 📈 Why Games Matter in AI and Data Education

게임은 학생들에게 매우 친숙한 매체이기 때문에,
추상적으로 느껴질 수 있는 인공지능과 데이터 개념을 구체적이고 흥미롭게 전달하는 데 매우 효과적입니다.

예를 들어:

* **NPC 행동** → 인공지능 의사결정 개념 이해
* **난이도 조절** → 사용자 맞춤형 시스템 이해
* **추천 시스템** → 데이터 기반 예측 이해
* **플레이 로그 분석** → 데이터 수집과 패턴 분석 이해
* **게임화** → 동기유발 설계와 행동 변화 유도 이해

따라서 게임은 단순한 놀이가 아니라,
AI와 데이터 교육을 위한 매우 훌륭한 학습 매개체가 될 수 있습니다.
